Feature overview

Support conversations without personal phone numbers.

Chat gives universities controlled messaging across their Intoto community. Students can contact the right person, staff can create groups, and conversations stay attached to university roles instead of scattered across personal apps.

What the client gets

When Active, Chat unlocks mobile Chats tab, web Chat sidebar, direct messaging, group creation, inboxes, message threads, read/unread states, and contextual sharing where other modules allow it. When inactive, chat surfaces are hidden.

FAQs

Chat questions clients usually ask.

Is Chat required for Community or Ambassador conversations?
Those modules can expose people and profiles, but in-product messaging requires Chat to be Active.
Can staff create group conversations?
Yes. Web chat supports group creation for relevant users and operational groups where permitted.
What happens when inactive?
Chat sidebar, mobile Chats tab, compose paths, inboxes, group creation, and in-product messaging are hidden.
